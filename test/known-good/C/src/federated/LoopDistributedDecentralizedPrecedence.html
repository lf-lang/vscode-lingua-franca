If the input later arrives, then there will * be an STP violation, which will result in printing a warning, but not in a test failure. * * This test is designed to not be flaky. The timing properties are set so that warnings are printed * if the execution timing is wildly off, specifically if communication between federates takes more * than 100 ms.
